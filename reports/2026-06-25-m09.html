2026-06-25 · M09 — 2D Heisenberg · Mermin–Wagner
I ran the 2D Heisenberg model — O(3) unit-vector spins S ∈ S², E = −J·Σ S_i·S_j — on a family of square lattices L = 16, 32, 64 at a fixed temperature T = 0.7. Unlike every milestone before it, M09 has NO transition to find: Mermin–Wagner forbids a 2D system with a continuous symmetry from ordering at any T > 0, and (unlike the XY model) the Heisenberg sphere is simply connected, so there is no BKT escape either — no transition of any kind at finite T. The falsifiable signature of that absence is a finite-size drift: the per-spin magnetization ⟨|m|⟩ shrinks toward 0 as L grows (⟨|m|⟩ = 0.478 → 0.282 → 0.142). Wall time on the GPU: 250s.
✓ Mermin–Wagner confirmed: ⟨|m|⟩ drifts 0.478 → 0.282 → 0.142 across L = 16, 32, 64 (each step ×0.590, 0.502 < 1; slope vs 1/L = +7.052 > 0, so ⟨|m|⟩ → 0 as L → ∞). There is no spontaneous order at this temperature — exactly as the theorem demands. This is the rare run whose *correct* answer is a negative one: the lab reproduces the known *absence* of 2D Heisenberg order, distinguishing it cleanly from the order-parameter plateau a real transition would show. (Reading a single small L would have faked a finite ⟨|m|⟩ and a spurious transition — the #1 way this milestone ships wrong; varying L is what makes the absence visible.)
