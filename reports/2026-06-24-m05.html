2026-06-24 · M05 — triangular-lattice 2D Ising
I ran the 2D Ising model on a *triangular* lattice (L=129) across a window straddling the transition and tracked the magnetic susceptibility χ(T). The triangular lattice is the square grid plus one diagonal — six neighbours per site, not four — so it is the same universality class as M01 but a different geometry, with its own *exact* critical temperature T_c = 4/ln 3 ≈ 3.6410. Because the triangular lattice is non-bipartite, this needed a 3-sublattice update, not the square checkerboard. The χ peak sits at T_c(L) = 3.675. Wall time on the GPU: 63s.
✓ χ-peak T_c(L) = 3.675 vs exact 4/ln3 = 3.6410 (rel. err 0.9%), and the specific-heat peak from the same run independently gives T_c(L) = 3.650. A new geometry reproduces its known answer: the triangular lattice's exact T_c falls right where the susceptibility peaks, on an engine that had to switch from the square checkerboard to a 3-sublattice update (the triangular lattice is non-bipartite). Same universality class as M01, different number — calibrated. (Small-L finite-size effects push the pseudo-critical peak slightly above the infinite-volume T_c; an L-extrapolation would sharpen it — see BACKLOG.)
