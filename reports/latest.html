2026-06-25 · M10 — antiferromagnetic Ising
I ran the *antiferromagnetic* 2D Ising model (J = −1) on an L=128 square lattice across a window straddling the transition. Flipping the coupling sign favours anti-aligned neighbours — the ground state is the checkerboard Néel state, not the aligned ferromagnet. On a bipartite lattice the sublattice gauge flip s_i → −s_i (on one colour) turns the antiferromagnet exactly into the ferromagnet, so the Néel temperature is Onsager's *same* exact T_c ≈ 2.2692. The order parameter is the *staggered* magnetization m_s = (1/N)Σ ε_i s_i (ε = (−1)^(x+y)); its susceptibility χ_s peaks at T_N(L) = 2.282. The uniform ⟨|m|⟩ stays ≈0 throughout — reading it would show nothing and look broken. Wall time on the GPU: 30s.
✓ Staggered χ_s-peak T_N(L) = 2.282 vs Onsager exact 2.2692 (rel. err 0.5%), and the specific-heat peak from the same run independently gives T_N(L) = 2.276. Flipping J to −1 doesn't break the engine: the antiferromagnet lands on Onsager's same critical temperature, read off the *staggered* order parameter (the uniform magnetization stays ≈0, the deliberate trap). The framework handles negative coupling cleanly — a calibration pass. The uniform magnetization stayed ≤ 0.003 across the sweep (the AFM carries no net moment — the staggered order does all the work). (Small-L finite-size effects push the pseudo-critical peak slightly above the infinite-volume T_N; an L-extrapolation would sharpen it.)
